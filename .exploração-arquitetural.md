## 2. PADRÕES DE MODAL/SHEET

### Modal Reutilizável (Radix UI Dialog)
**Arquivo:** `/components/ui/Modal.tsx`

```tsx
interface Props {
  open: boolean
  onClose: () => void
  title: string
  children: React.ReactNode
  className?: string
}
```

**Características:**
- Baseado em `@radix-ui/react-dialog` (v1.1.15)
- Overlay com `bg-black/60`
- Conteúdo: `max-w-lg max-h-[90vh] overflow-y-auto`
- Styling: `bg-navy border border-white/10 rounded-2xl`
- Header sticky com título e botão de fechar (X)
- Z-index: 40 (overlay), 50 (conteúdo)

**Uso Exemplo:** `NewLeadModal.tsx` e `InscritoModal.tsx`

### Bottom Sheet (Custom Div)
**Arquivo:** `DespesasPage.tsx` (linhas 211-310)

```tsx
{sheetOpen && (
  <>
    <div className="fixed inset-0 bg-black/60 z-40" />
    <div className="fixed bottom-0 left-0 right-0 bg-footer border-t border-white/10 
                    rounded-t-2xl z-50 p-4 max-h-[90vh] overflow-y-auto">
```

**Características:**
- Overlay fixo: `bg-black/60 z-40`
- Sheet: `fixed bottom-0 ... rounded-t-2xl z-50`
- Cor de fundo: `bg-footer` (#08111F)
- Max height: `90vh`
- Padrão: `onClick={() => setSheetOpen(false)}` no overlay

**Outro exemplo:** `MoverLeadSheet.tsx` (linhas 30-54)

## 9. RECOMENDAÇÕES PARA NOVOS COMPONENTES

✅ **Fazer:**
1. Usar `Modal` de `/components/ui/Modal.tsx` para diálogos
2. Usar bottom sheet custom para ações em mobile (pattern do `DespesasPage`)
3. Aplicar classes de `.btn-primary`, `.btn-secondary`, `.input-field`
4. Usar `font-display` para títulos, `font-sans` para corpo
5. Manter z-index: 40 (overlay), 50 (conteúdo modal)
6. Respeitar breakpoint `md:` para layout responsivo
7. Usar `lucide-react` para ícones
8. Usar `sonner` para toasts
9. Extrair lógica em hooks customizados
10. Colorir com `orange`, `navy`, `muted` do theme

❌ **Evitar:**
- Criar novos componentes de modal (use `Modal.tsx`)
- Usar inline styles (use Tailwind ou classes reutilizáveis)
- Z-index aleatórios (padrão: 40, 50, 20 para sidebar)
- Cores hardcoded (usar tema do tailwind.config.js)
- Duplicar lógica (extrair em hooks)

## PRÓXIMOS PASSOS

Para criar componentes de Receitas/Inscrições que sigam o padrão:

1. **Página:** Criar `RecebitasPage.tsx` no style de `DespesasPage.tsx`
   - Header com título e CTA
   - Filtros (similar aos de Despesas)
   - Lista scrollável
   - Bottom sheet para criar

2. **Modal/Sheet:** Usar `Modal.tsx` ou bottom sheet customizado
   - Manter coerência visual com `DespesasPage`
   - Usar mesmas classes de input/button

3. **Tipos:** Adicionar à `lib/types.ts` se necessário

4. **Hooks:** Criar `useReceitas()` similar a `useDespesas()`

5. **Styling:** Respeitar cores, fonts, espacos já definidos
